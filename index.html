Happy Valentines Day! This Valentines Day I wasn't able to see my girlfriend, so I made a little chrome extension for her!

The extension replaces the newtab page of the Chrome browser. Instead of the default page, it displays a page with a random dog (she loves dogs!) as the background and a random message (which is personalized and preset).

Click here to see what replaces the newtab page! You can refresh to re-randomize the text and background picture!

This game was made using the Unity game engine along with C#. It is a single player, FPS (first person shooter) game.

It features AI enemies that patrol, chase, and attack the player, a spawner that randomly spawns enemies, player shooting and reload animations, an informative HUD (heads up display), a main and pause menu system, a final boss, and some game sounds recorded by yours truly!

I had a really rough time starting the project, but I definately learned a lot over the course of creating this project! I had a lot of fun creating this game and I hope you have fun playing it too!

Technologies:
  • Unity game engine
  • C#
Challenges and things I learned:
  • Creating a player controller that felt smooth to control and a HUD to keep the player informed
    • this includes making animations for shooting and reloading the gun, which I think turned out to look really cool
  • Creating AI enemies that didn't bug out and crash the game
    • there were an unbelieveable number of bugs related to the enemies
    • one of the buggiest was probably was the enemies' pathfinding and interacting with the map
    • another big challenge of this was creating enemies that shoot projectiles at the player, and making the projectiles behave properly as well
  • Creating a spawner that spawns those enemies onto the map in random locations
  • Creating a main and pause menu that functioned properly (this was also surprisingly really prone to bugs)
  • Creating a map where the game could take place
  • Adding quality of life features to help the player know what they need to do next
  • Finding art that fit what I envisioned
